
# About Me – Personal Webpage

This is a personal About Me webpage built with [Next.js](https://nextjs.org), [TypeScript](https://www.typescriptlang.org/), and [Tailwind CSS](https://tailwindcss.com/). It is designed to showcase your profile, skills, experience, and contact information—perfect for linking from your GitHub profile.


## Features

- Modern, responsive design
- About Me section with profile image and bio
- Skills and experience sections
- Contact information with links
- Built with Next.js App Router, TypeScript, and Tailwind CSS

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view your About Me page.

### Customization

- Update your name, bio, skills, experience, and contact info in `src/app/about/page.tsx`.
- Replace `public/profile-placeholder.png` with your own profile photo.


## Deployment

You can deploy this site to [Vercel](https://vercel.com/) or any platform that supports Next.js.
